## Descrição do Projeto
Um Sistema de Gestão de Compras desenvolvido em Vue.js para a disciplina de Frameworks Modernos. O objetivo é gerenciar uma lista de produtos em tempo real usando reatividade


## Como rodar o projeto
Para executar este projeto localmente:

1. Instale as dependências:
   npm install

2. Iniciar o servidor de desenvolvimento:
  npm run dev
